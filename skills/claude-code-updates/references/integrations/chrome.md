# Chrome Integration

Connect Claude Code to Chrome for web automation.

## Capabilities

| Feature | Description |
| --- | --- |
| Page navigation | Open URLs, click links |
| Form interaction | Fill forms, submit |
| Console access | Read console logs |
| Screenshot | Capture page screenshots |
| Data extraction | Extract page content |

## Setup

1. Install Chrome extension
2. Enable browser connection in Claude Code
3. Grant necessary permissions

## Use Cases

- Test web applications
- Debug with console logs
- Automate form filling
- Extract data from pages
