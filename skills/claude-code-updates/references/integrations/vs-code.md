# VS Code Integration

Use Claude Code within VS Code.

## Features

| Feature | Description |
| --- | --- |
| Inline diffs | See changes in editor |
| @-mentions | Reference files and symbols |
| Plan review | Review before applying |
| Keyboard shortcuts | Quick access |

## Installation

1. Open VS Code Extensions
2. Search for 'Claude Code'
3. Click Install

## Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| `Cmd+Shift+P` | Open Claude Code |
| `Cmd+L` | Send selection to Claude |

## @-mentions

Reference context in prompts:

- `@file.ts` - Reference a file
- `@symbol` - Reference a function/class
- `@selection` - Current selection
